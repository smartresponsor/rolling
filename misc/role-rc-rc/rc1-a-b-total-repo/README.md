# role-rc-rc / rc1-a-b-total-repo

Сводный рабочий пакет (репозиторий) для RC1..RC2 компонента Role.

- `kits/` — распакованные наборы A (RC1 интеграция) и B (надёжность/перф).
- `repo/` — рабочая копия кода под git.

## Порядок

1) `git init` в `repo/`, первый коммит.
2) Интегрируй по очереди киты (см. `repo/tools/integrate-kit.*`), коммить после каждого.
3) Запускай смоук/бенчи из `repo/tools` и `repo/bin`.
