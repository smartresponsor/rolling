# SDK workspace

This directory is the canonical home for language SDK assets that are shipped with the Rollin workspace.

Subtrees:
- `php/` — PHP SDK sources used by Composer autoload and packaging.
- `js/` — JavaScript SDK client and typings.
- `java/` — Java SDK sources and build metadata.

Top-level package manifests, release notes, checksums, and runbooks were moved to `report/package/sdk/` so that `misc/` stays SDK-only.
